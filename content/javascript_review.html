Web Dev Notes - Javascript Review

Summary:

Javascript is the front-end language of the web, central to coding in a browser. Via Node.js, it has also become a viable server-side scripting language, indicating a growing reliance on Javascript for all forms of web development.

Here I teach-to-learn a variety of new and old Javascript ideas.

querySelector

When I first started learning Javascript, I was initially taught to access an HTML element like this: 

document.getElementById("element");

(where element here is the ID assigned to the HTML element in question)

I found this a wonderfully persuasive argument for switching to always using jQuery and its rather simpler version: 

$("#element")

It appears that another, slightly less "clunky" version is also available in plain Javascript: 

 document.querySelector("p.class")

Now, ever since I started learning HTML at 15, it has seemed that web standards have advanced and changed for almost arbitrary reasons. It was easy for me to assume that this was the equivalent of using <strong> instead of <b> - functionally the same, though a result of a change in ideological approach.

However, both getElementbyID and querySelector are "up to date" Javascript and do in fact vary more from one another than one might expect.
