Web Dev Notes - Classes in JavaScript

Summary:

Classes are the primary object oriented programming feature. Here's how they can be implemented in JavaScript.

What are classes?

Outside of JavaScript, classes are often an inbuilt part of a programming language. They are used to create a multitude of objects on the same pattern, which can all do the same things. A common analogy is that a class is like a blueprint to build an object.

JavaScript doesn't have inbuilt classes, but OOP-style programming can still be done by making your own classes, implemented through a variety of methods. These create new objects and give them properties. The functions used to so are called "constructors"

How do you build a class?

classes are created in a very similar way to object decorators, except that instead of adding properties to an already existing object, constructors create a whole new object. Conventionally, a class is named with a capitalised noun.
