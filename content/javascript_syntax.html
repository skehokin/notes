Javascript Syntax

Summary:

It's important to keep the knowledge inside your head rather than on the internet.

document.querySelector(argument)
finds the first element matching the argument: a string resembling a CSS selector
document.querySelector(argument)
finds ALL elements matching the argument: a string resembling a CSS selector
Node.textContent
what it says on the tin
Element.getAttribute(argument)
this gets the value of an HTML attribute. the argument is the name of the attribute you want to change.
Element.getAttribute(argument1, argument2)
this sets the value of an HTML attribute. the first argument is the name of the attribute you want to change. The second argument is the new value.
