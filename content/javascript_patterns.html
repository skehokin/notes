Web Dev Notes - JavaScript Design Patterns

Summary:

Design Patterns are established (maybe even conventional) solutions for solving similar problems in (hopefully) the best way.

Object Decorator Pattern

Decorator functions take an object and add certain properties and methods to that object. The methods inside given to every object can point to a single (global) function definition, but it's helpful to make them point to their own version of the function within the decorator function, thus avoiding using the "this" parameter, and keeping all relevant code together.

The name of a decorator function is often an adjective.
